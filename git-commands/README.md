# comandos-git

- git init
  Inicia um repósitorio git no código.
  
- git add <nome do arquivo>
  Adiciona um arquivo ao "staged", que é uma sessão de preparação para um commit.

- git diff
  Compara diferenças do código atual para o código que está commitado.

- git status
  Mostra como está o status da branch, se há algo para ser comitado, o que está no stage e etc.

- git commit -m "mensagem da commit"
  commita os arquivos em stage com a mensagem da sua escolha.

- git log
  Mostra o histórico de commits.

- git branch
  Mostra a branch selecionada e as disponiveis.

- git checkout <nome da branch>
  Seleciona a branch da sua escolha.

- git push origin <nome da branch>
  Sobe a branch escolhida para o repositorio remoto.

- git pull origin <nome da branch>
  Puxa a branch selecionada do repositorio remoto para o local.

- git merge 
  Mescla branch atual com a desejada.

- git branch -b <nome>
  Cria uma branch a partir da branch atual.

- git remote add <nome> <url>
  Add um novo repositorio remoto

- git fetch
  atualiza o historico local de acordo com o historico remoto, trazendo branchs.
